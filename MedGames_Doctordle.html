Como jogar

Cada rodada apresenta um caso clínico com um diagnóstico oculto. Seu objetivo é adivinhar qual é o diagnóstico correto usando o menor número de dicas possível.

Regras do jogo

  1. Na tela, você verá uma dica de cada vez sobre o caso clínico.
  2. Digite seu palpite no campo de texto e clique em "Adivinhar" (ou pressione Enter).
  3. Se acertar, o jogo termina e você ganha!
  4. Se errar, uma nova dica é revelada e você tenta novamente.
  5. Você tem no máximo 6 tentativas (uma para cada dica).
  6. Se errar todas as 6 dicas, o diagnóstico correto é revelado.

Dicas importantes

Dica 1 é a mais genérica. A Dica 6 é a mais específica. Quanto menos dicas você usar para acertar, melhor!
Você pode começar a digitar no campo de palpite e uma lista de sugestões aparecerá automaticamente com os diagnósticos possíveis. Use as setas do teclado ou clique em uma sugestão para selecioná-la.
A comparação é flexível: não importa se você usa maiúsculas, minúsculas ou espaços extras. O importante é que o texto corresponda ao diagnóstico.
